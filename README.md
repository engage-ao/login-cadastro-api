# Sistema de Login e Cadastro

Projeto desenvolvido durante o estágio, responsável pelo módulo de autenticação.

##  Estrutura do Projeto

Estagio/
┣ frontend/ → Angular
┗ backend/ → Flask + SQLAlchemy

## Tecnologias Utilizadas
- Angular
- Node.js
- Flask
- SQLAlchemy
- Python

##  Funcionalidades
- Cadastro de usuários
- Login com validação
- API REST com Flask
- Integração frontend e backend
